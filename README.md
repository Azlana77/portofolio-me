# Website Portofolio Azis Maulana

Website portofolio pribadi Azis Maulana, mahasiswa Teknik Informatika dengan minat pada IoT, elektronika, database, dan pengembangan sistem.

## Tech Stack

* React
* Vite
* Tailwind CSS v4
* lucide-react

## Cara Menjalankan

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy ke Vercel

1. Push project ke GitHub.
2. Buka Vercel.
3. Pilih Add New Project.
4. Import repository GitHub.
5. Framework preset pilih Vite.
6. Build command: npm run build.
7. Output directory: dist.
8. Klik Deploy.
